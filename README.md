# Haunted-House
A haunted house scene built withThree.js native geometries and Poly Haven's textures.

## Run Project Locally

Clone the project

```bash
  git clone https://github.com/BasilOmsha/Haunted-House.git
```

Go to the project directory

```bash
  cd Haunted-House
```

Install the packages

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
